# Algorithms and Data Structures in Go

List of algorithms:
  - Linear Search -> $O(n)$
  - Binary Search -> $O(log(n))$
  - Bubble Sort -> $O(n^2)$
  - Two Crystal Ball Problem -> $O(\sqrt{ N })$

List of Data Structures: 
  - Binary Tree
  - Doubly Linked List 
  - Queue
  - Ring Buffer
  - Singly Linked List
  - Stack
