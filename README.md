# Algorithms and Data Structures in Go

List of algorithms:
  - Linear Search -> $O(n)$
  - Binary Search -> $O(log(n))$
  - Bubble Sort -> $O(n^2)$
  - Two Crystal Ball Problem -> $O(\sqrt{ N })$
  - Breadth-First Search (BFS) in Binary Tree -> $O(V + E)$
  - Maze Solver -> Complexidade depende do algoritmo
